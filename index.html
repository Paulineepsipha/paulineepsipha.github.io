The sole purpose of this project is to provide all the students and the people with such a network of auto rickshaws so that it eases the people to travel whenever and wherever required. While the driver side the issue is different. All the driverâ€™s are not getting the long route trips. Sometimes it becomes hard to find passengers. So counting the problems faced by both sides we have decided to make a mobile application using which the passengers can book the auto from the application as per his/her convenience. As well as we will also make application for driver application in which the driver will get the request from passenger for the ride and he can accept or deny the request as per his schedule. Sentiment Analysis - Know The Opinion
Android Application

In modern world, Sentiment Analysis or opinion mining is the computational study of people opinions, sentiments, attitudes, and emotions expressed in written language. It is one of the most active research areas in natural language processing and text mining. Its popularity is mainly due to two reasons. First, it has a wide range of applications because opinions are central to almost all human activities and are key influencer of our behaviors. Whenever we need to make decision, we want to hear othersâ€™ opinions. Second, it presents many challenging research problems, which had never been attempted before the year 2000.

About application


This application is made to discover the opinion of public on particular topic. We just have to input the topic which we want to analyse. Maximum 100 recent tweets would be analysed and tweets would be 7 days old. The application works with Naive Bayes Classifier and generates the sentiment in positive and negative polarity. Similar visualisation chart- Bar chart and Pie chart by Emotions are still in development.




User need not to have a twitter account for application usage.
